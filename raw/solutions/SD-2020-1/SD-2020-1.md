# 考題編號：SD-2020-1

**主分類：** `SD-U1-1` 結構動力基本性質及原理
**副分類：** `SD-U3-2` 隔減震原理
**分析方法：** 概念題（推導＋圖形說明）
**標籤：** `SDOF` `動力放大因子` `被動傳導率` `消能減震` `基礎隔震` `阻尼比` `頻率比` `穩態反應` `傳導率曲線` `隔震原理`

---

## 1. 原始題目重述 (Problem Restatement)

單自由度線彈性系統：

- **動力放大因子（Dynamic Magnification Factor, DMF）：** 受簡諧外力作用時，穩態位移振幅與相同大小**靜態力**所產生位移之比值
- **被動傳導率（Passive Transmissibility, TR）：** 受地表簡諧加速度擾動時，結構**絕對加速度**振幅與地表加速度振幅之比值

要求：
1. 推導 DMF 與 TR 的公式
2. 畫出 $\xi = 0.01, 0.05, 0.20$ 的曲線
3. 以 DMF 解釋消能減震原理
4. 以 TR 解釋基礎隔震原理

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
- DMF 描述**直接力激振**的放大效果（消能設備安裝在結構上，改變有效阻尼）
- TR 描述**地基激振**的傳遞效果（隔震墊降低地基運動傳遞到結構）

**出題者測驗能力：**
1. 能否正確推導兩個無量綱比值
2. 能否辨識曲線形狀特徵（共振峰、大 $r$ 時的收斂行為差異）
3. 能否連結曲線特徵到工程應用（消能 vs 隔震）

**關鍵洞察：** DMF 和 TR 的數學形式幾乎相同，但物理意涵不同。最重要的差異在於 **TR 在大 $r$ 時，大阻尼反而使傳導率偏高**（與 DMF 不同），這正是隔震設計的核心矛盾。

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**四大陷阱：**

| # | 陷阱 | 正確處理 |
|---|------|---------|
| 1 | DMF 與 TR 公式相混淆 | DMF 分子為 1；TR 分子含 $\sqrt{1+(2\xi r)^2}$ |
| 2 | 隔震設計中認為阻尼越大越好 | $r > \sqrt{2}$ 時，TR 隨阻尼增加而**增大**（不利） |
| 3 | 消能減震以為只降共振 | 消能同時對所有頻率有效（增加分母阻尼項） |
| 4 | 繪圖時未標出 $r = \sqrt{2}$ 的交叉點 | TR 曲線在 $r=\sqrt{2}$ 時與阻尼無關，需標注 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標

推導 DMF 和 TR，以圖形說明消能減震與基礎隔震原理

### 本題關鍵公式（依推導順序）

$$\text{Step 1: EOM（直接力）} \quad m\ddot{x} + c\dot{x} + kx = F_0\sin(\Omega t)$$

$$\text{Step 2: EOM（基礎激振）} \quad m\ddot{u} + c\dot{u} + ku = -m\ddot{x}_g = m\ddot{x}_g^{amp}\sin(\Omega t)$$

$$\text{Step 3: DMF} \quad D = \frac{1}{\sqrt{(1-r^2)^2 + (2\xi r)^2}}$$

$$\text{Step 4: 絕對加速度} \quad \ddot{x}_{abs} = \ddot{u} + \ddot{x}_g$$

$$\text{Step 5: TR} \quad TR = \sqrt{\frac{1+(2\xi r)^2}{(1-r^2)^2+(2\xi r)^2}}$$

### L1：題目直接給定

| 符號 | 說明 |
|------|------|
| $\xi$ | 0.01、0.05、0.20 三條曲線 |
| $r = \Omega/\omega_0$ | 頻率比（橫軸） |

### L2：需知識點推導

| 步驟 | 內容 | 卡關? |
|------|------|------|
| 穩態特解 | $x_p = X\sin(\Omega t - \phi)$，求 $X$ | |
| 靜位移 | $x_{st} = F_0/k$ | |
| DMF | $D = X/x_{st}$ | |
| 絕對加速度 | 相對位移加回地動加速度 | |
| TR 分子 | 彈簧力＋阻尼力傳遞到地面：$T = F_{trans}/F_0$ | |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|------|
| TR 分子含 $2\xi r$ 項的物理原因 | 阻尼器傳遞「阻尼力」到地面，頻率越高（$r$ 越大）阻尼力越大，造成 TR 在大 $r$ 時隨阻尼增加 | |
| $r > \sqrt{2}$ 為隔震區的物理意義 | 此時 $1-r^2 < -1$，彈簧力的傳遞已被慣性主導，結構不再跟隨地面 | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### (一) 動力放大因子（DMF）推導

**系統方程式（外力激振）：**

$$m\ddot{x} + c\dot{x} + kx = F_0\sin(\Omega t)$$

除以 $m$，令 $\omega_0^2 = k/m$，$2\xi\omega_0 = c/m$：

$$\ddot{x} + 2\xi\omega_0\dot{x} + \omega_0^2 x = \frac{F_0}{m}\sin(\Omega t)$$

**穩態特解：**

$$x_p(t) = X\sin(\Omega t - \phi)$$

代入方程式，比較係數：

$$X = \frac{F_0/k}{\sqrt{(1-r^2)^2 + (2\xi r)^2}}, \qquad r = \frac{\Omega}{\omega_0}$$

$$\phi = \arctan\!\left(\frac{2\xi r}{1-r^2}\right)$$

**動力放大因子：**

$$\boxed{D = \frac{X}{F_0/k} = \frac{1}{\sqrt{(1-r^2)^2 + (2\xi r)^2}}}$$

---

### (二) 被動傳導率（Passive Transmissibility）推導

**系統方程式（地基激振）：**

設地表位移 $x_g = X_g\sin(\Omega t)$，相對位移 $u = x - x_g$：

$$m\ddot{u} + c\dot{u} + ku = -m\ddot{x}_g = m\Omega^2 X_g\sin(\Omega t)$$

這與外力激振形式相同，稱為「等效力激振」。相對位移振幅：

$$U = \frac{r^2 X_g}{\sqrt{(1-r^2)^2 + (2\xi r)^2}}$$

**絕對加速度振幅：**

結構絕對位移 $x = u + x_g$，絕對加速度振幅：

傳遞到結構的力（來自彈簧與阻尼器）= $kx + c\dot{x}$（稳态下）。

利用向量合成：絕對加速度振幅 $= \omega_0^2 X_{abs}$，其中

$$X_{abs} = \sqrt{U^2 + X_g^2 + 2U X_g\cos\psi}$$

等價方法（直接推導）：

絕對加速度由以下公式得到（利用 $x = u + x_g$）：

$$m\ddot{x} = -(ku + c\dot{u}) = -(kx + c\dot{x}) + (kx_g + c\dot{x}_g)$$

注意：地面傳遞至結構的力（base shear）= $ku + c\dot{u}$，其振幅：

$$F_{trans} = k\sqrt{U^2\left(1 + (2\xi r)^2\cdot\frac{r^2}{1}\right)} = kX_g\frac{\sqrt{1+(2\xi r)^2}}{\sqrt{(1-r^2)^2+(2\xi r)^2}}$$

更直接的推導：絕對加速度振幅 $= (F_{trans}/m)$，即：

$$\frac{\ddot{X}_{abs}}{\Omega^2 X_g} = \sqrt{\frac{1+(2\xi r)^2}{(1-r^2)^2+(2\xi r)^2}}$$

**被動傳導率（絕對加速度比值）：**

$$\boxed{TR = \frac{|\ddot{X}_{abs}|}{|\ddot{X}_g|} = \sqrt{\frac{1+(2\xi r)^2}{(1-r^2)^2+(2\xi r)^2}}}$$

---

### (三) 曲線特徵說明

**DMF 曲線特徵（圖一）：**

| 區域 | 行為 | 物理意涵 |
|------|------|---------|
| $r \ll 1$（低頻） | $D \approx 1$ | 準靜態：結構跟隨外力 |
| $r = 1$（共振） | $D = 1/(2\xi)$ | 共振放大，阻尼越小放大越劇烈 |
| $r \gg 1$（高頻） | $D \to 0$ | 高頻力無法驅動結構 |
| 阻尼效果 | 所有 $r$ 皆有益 | 阻尼越大，DMF 越小（共振區最明顯） |

**TR 曲線特徵（圖二）：**

| 區域 | 行為 | 物理意涵 |
|------|------|---------|
| $r = 0$ | $TR = 1$ | 結構完全跟隨地面 |
| $r = 1$（共振） | $TR = 1/(2\xi)$（近似） | 共振放大 |
| $r = \sqrt{2}$ | $TR = 1$（與阻尼無關） | 所有阻尼曲線交於此點 |
| $r > \sqrt{2}$ | $TR < 1$（隔震區） | 結構加速度小於地面加速度，**大阻尼反而不利（TR 較大）** |

*（$r = \sqrt{2}$ 交叉點的推導：令 $TR = 1$，化簡得 $r = \sqrt{2}$）*

---

### (四) 消能減震原理（以 DMF 解釋）

**原理：** 在結構上安裝消能裝置（黏滯阻尼器、金屬降伏阻尼器等），增加系統**等效阻尼比** $\xi_{eff}$。

由 DMF 曲線可見：
- 在**共振區**（$r \approx 1$），阻尼從 $\xi = 0.01$ 增加到 $\xi = 0.20$，DMF 從約 50 降至 2.5（降低約 20 倍）
- 在**非共振區**，阻尼增加同樣使 DMF 降低，但效果較不顯著

$$D_{max} = \frac{1}{2\xi} \quad \text{（共振時）} \Rightarrow \xi \uparrow \Rightarrow D_{max} \downarrow$$

**消能減震策略：**
- 結構物振動週期多落在 $r \approx 1$ 附近（設計地震頻率接近結構自然頻率）
- 安裝消能設備使 $\xi_{eff}$ 從 3~5% 提高至 15~30%，可顯著降低結構反應

---

### (五) 基礎隔震原理（以 TR 解釋）

**原理：** 在基礎與上部結構之間安裝隔震層（如 LRB 鉛心橡膠墊），大幅**降低系統自然頻率**（延長週期 $T$），使頻率比 $r = T_g/T_{structure} \gg \sqrt{2}$（進入隔震區）。

由 TR 曲線可見：
- 當 $r = 3$（隔震後典型頻率比），$\xi = 0.05$ 時 $TR \approx 0.11$（傳遞率僅 11%）
- 上部結構受到的加速度僅為地面加速度的 11%，大幅降低地震力

$$TR \approx \frac{1}{r^2}\sqrt{1+(2\xi r)^2} \;\text{（大 r 近似）}$$

**隔震設計的關鍵矛盾：**

$$\text{當 } r > \sqrt{2} \text{：} TR \text{ 隨 } \xi \text{ 增加而增大（阻尼不利）}$$

∴ 隔震設計不應追求高阻尼——應以**低阻尼延長週期**為主，只在需要控制位移時才適當加入阻尼（通常 $\xi_{iso} = 10 \sim 20\%$，已足夠控制隔震層位移）。

---

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**爭議點：消能減震 vs 隔震的本質差異**

| 對比項目 | 消能減震 | 基礎隔震 |
|---------|---------|---------|
| 作用機制 | 增加阻尼（$\xi\uparrow$），降低 DMF | 延長週期（$r\uparrow$），降低 TR |
| 對應曲線 | DMF | TR |
| 地震力作用位置 | 外力激振（結構受風或機械振動） | 地基激振（地震） |
| 阻尼的效果 | 越大越好（共振區） | 在隔震區（$r>\sqrt{2}$）越大越不利 |
| 適用結構 | 中高層、改造加固 | 低矮重要建築（醫院、核電站） |

**進階：TR 分子含 $2\xi r$ 項的物理意涵**

阻尼器是「硬連接」，它傳遞與速度成正比的力。在高頻激振（$r$ 大）時，阻尼力可以很大，反而使更多力傳遞到結構，因此 $r > \sqrt{2}$ 時高阻尼不利於隔震效果。這也是為何隔震支承的阻尼不宜過大（通常 $\leq 20\%$）。
