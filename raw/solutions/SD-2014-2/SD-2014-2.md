## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
- LRB 的雙線性遲滯行為需以「等效線性化」轉化為等效彈簧 + 等效阻尼系統
- 等效週期 $T_{eD}$ 依賴設計位移 $D_D$，設計位移又依賴 $T_{eD}$，需迭代收斂
- 阻尼修正係數 $B$ 是將 5% 阻尼反應譜換算到隔震系統實際阻尼 $\xi_{eD}$ 的橋梁

**出題者意圖：**
- 測驗考生能否完整推導 $K_{eD}$（等效割線勁度）的公式
- 測驗考生能否推導 $A_{TD}$（遲滯迴圈消能面積）
- 測驗考生能否寫出清晰的迭代流程（步驟分明，公式完整）

**關鍵陷阱：**
1. 等效勁度是**割線勁度**（secant stiffness），不是初始勁度 $k_o$
2. $A_{TD}$ 是遲滯迴圈**面積**（能量消散），不是剛度乘位移
3. $B$ 因子選 $B_1$（長週期），非 $B_S$（短週期），因隔震建築 $T_{eD} \gg 1$ sec

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**解題路線（等效線性化迭代法）：**

```
假設初始 D_D（或初始 K_eD）
         ↓
 Step 1  建立 LRB 雙線性模型（單支 + 整體系統）
         ↓
 Step 2  計算整體有效勁度 K_eD（割線勁度）
         ↓
 Step 3  計算等效週期 T_eD
         ↓
 Step 4  計算遲滯迴圈面積 A_TD
         ↓
 Step 5  計算等效阻尼比 ξ_eD
         ↓
 Step 6  由 ξ_eD 查表得阻尼修正係數 B = B_1
         ↓
 Step 7  由反應譜計算新的 D_D
         ↓
 D_D 是否收斂？ ──否──→ 回 Step 2
         ↓ 是
   收斂 → 輸出 D_D
```

**關鍵陷阱：**

| # | 陷阱 | 應對 |
|---|------|------|
| 1 | 割線勁度公式 | $K_{eD} = F(D_D)/D_D$（非 $K_o$） |
| 2 | $A_{TD}$ 只在 $D_D > D_y$ 時有效 | 先確認 $D_D > D_y = F_y/k_o$ |
| 3 | 長週期時 $S_{aD}$ 隨週期遞減 | 每次迭代須重新讀取對應 $T_{eD}$ 的 $S_{aD}$ |
| 4 | $B$ 因子來源 | $B = B_1$ 從規範表格（以 $\xi_{eD}$ 查表），非自行設定 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標

計算設計位移 $D_D$（隔震系統在設計地震下的最大水平位移）。

### 本題關鍵公式（依計算順序）

$$\text{Step A:}\quad D_y = \frac{F_y}{k_o} \quad \text{（單支 LRB 降伏位移）}$$

$$\text{Step B:}\quad K_{eD} = n\!\left[\frac{F_y(1-\alpha)}{D_D} + \alpha k_o\right] \quad \text{（整體有效割線勁度）}$$

$$\text{Step C:}\quad T_{eD} = 2\pi\sqrt{\frac{W}{\boxed{K_{eD}} \cdot g}}$$

$$\text{Step D:}\quad A_{TD} = 4n \cdot F_y(1-\alpha)\!\left(D_D - \frac{F_y}{k_o}\right) \quad \text{（整體遲滯迴圈面積）}$$

$$\text{Step E:}\quad \xi_{eD} = \frac{1}{2\pi}\cdot\frac{\boxed{A_{TD}}}{\boxed{K_{eD}}\cdot D_D^2}$$

$$\text{Step F:}\quad B = B_1(\xi_{eD}) \quad \text{（查規範阻尼修正表）}$$

$$\text{Step G:}\quad D_D^{(\text{new})} = \frac{g}{4\pi^2}\cdot\frac{S_{aD}(\boxed{T_{eD}})\cdot \boxed{T_{eD}}^2}{\boxed{B}}$$

### L1：題目直接給定

| 符號 | 說明 |
|------|------|
| $W$ | 上部結構總重 |
| $n$ | LRB 數量（= 柱數） |
| $k_o$ | 單支 LRB 初始勁度 |
| $F_y$ | 單支 LRB 降伏強度 |
| $\alpha$ | 降伏後勁度比 |
| $S_{aD}$ | 設計地震 5% 阻尼反應譜（函數） |
| $B_S, B_1$ | 阻尼修正係數（短/長週期，查表） |

### L2：需知識點推導

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|
| $D_y$ | $F_y/k_o$（降伏位移） | |
| $K_{eD}$ | 割線勁度 $= F(D_D)/D_D$，代入雙線性公式 | |
| $A_{TD}$ | 雙線性遲滯迴圈面積公式（平行四邊形近似） | |
| $\xi_{eD}$ | 等效阻尼比（能量法） | |
| $T_{eD}$ | 等效線性化週期 | |
| $B_1$ | 規範阻尼修正表（$\xi_{eD}$ → $B_1$） | |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|-------|
| 割線勁度 vs 初始勁度 | 等效線性化使用割線勁度（降伏後的平均斜率），非初始切線勁度 | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 【前置】雙線性 LRB 模型建立

**單支 LRB 力-位移關係（雙線性）：**

$$F_\text{LRB}(u) = \begin{cases} k_o \cdot u & u \leq D_y \\ F_y + \alpha k_o (u - D_y) & u > D_y \end{cases}$$

降伏位移：

$$D_y = \frac{F_y}{k_o}$$

**$n$ 支 LRB 並聯（整體隔震系統）：**

- 整體初始勁度：$K_o = n k_o$
- 整體降伏強度：$F_Y = n F_y$
- 整體降伏後勁度：$\alpha K_o = n \alpha k_o$

---

### Step 1：假設初始試算設計位移 $D_D^{(0)}$

**判斷：** 設計位移 $D_D$ 必須大於降伏位移 $D_y = F_y/k_o$（隔震器才能充分消能）。

選取初始試算值，例如假設 $D_D^{(0)}$（可從規範建議初始值或工程經驗估算）。

---

### Step 2：計算整體有效割線勁度 $K_{eD}$

在位移幅值 $D_D$ 下，單支 LRB 的割線（secant）勁度：

$$k_{eff} = \frac{F_y + \alpha k_o(D_D - D_y)}{D_D} = \frac{F_y(1-\alpha)}{D_D} + \alpha k_o$$

整體隔震系統有效勁度（$n$ 支並聯）：

$$\boxed{K_{eD} = n \left[\frac{F_y(1-\alpha)}{D_D} + \alpha k_o\right]}$$

> **策略註解：** $K_{eD}$ 包含兩項：$\alpha k_o$（降伏後斜率貢獻）+ $F_y(1-\alpha)/D_D$（遲滯迴圈的等效勁度貢獻）；隨 $D_D$ 增大，$K_{eD}$ 遞減（割線勁度降低）。

---

### Step 3：計算等效週期 $T_{eD}$

上部結構視為剛體（集中質量 $W/g$），隔震系統有效勁度 $K_{eD}$：

$$\boxed{T_{eD} = 2\pi\sqrt{\frac{W}{K_{eD} \cdot g}}}$$

---

### Step 4：計算遲滯迴圈面積 $A_{TD}$（整體消能能量）

雙線性系統一個完整遲滯迴圈（往返一次）的能量消散面積（平行四邊形）：

單支 LRB 消能面積：
$$a_{TD} = 4\cdot(F_y - \alpha k_o D_y)\cdot(D_D - D_y) = 4F_y(1-\alpha)\!\left(D_D - \frac{F_y}{k_o}\right)$$

$n$ 支並聯整體消能面積：

$$\boxed{A_{TD} = 4n F_y(1-\alpha)\!\left(D_D - \frac{F_y}{k_o}\right)}$$

> **推導說明：** 雙線性迴圈為平行四邊形，底邊 $= 2(D_D - D_y)$，高 $= 2Q_d$，其中特性強度 $Q_d = F_y - \alpha k_o D_y = F_y(1-\alpha)$，故面積 $= 4 Q_d (D_D - D_y)$。

---

### Step 5：計算等效阻尼比 $\xi_{eD}$

等值阻尼比由能量法定義（消能能量 / 最大彈性應變能 × $\frac{1}{2\pi}$）：

$$\boxed{\xi_{eD} = \frac{1}{2\pi}\cdot\frac{A_{TD}}{K_{eD}\cdot D_D^2}}$$

代入 Step 2、4 的公式，可顯示：

$$\xi_{eD} = \frac{1}{2\pi}\cdot\frac{4n F_y(1-\alpha)(D_D - F_y/k_o)}{n[F_y(1-\alpha)/D_D + \alpha k_o]\cdot D_D^2}$$

> **物理意義：** $K_{eD} D_D^2/2$ 為最大彈性應變能，$A_{TD}/(4\pi)$ 為等效阻尼消能，兩者之比乘以 $\frac{1}{2}$ 即 $\xi_{eD}$。

---

### Step 6：由 $\xi_{eD}$ 確定阻尼修正係數 $B$

隔震建築等效週期 $T_{eD}$ 通常 $\gg 0.6$ sec（長週期範圍），故選用**長週期**阻尼修正係數：

$$B = B_1(\xi_{eD}) \quad \text{（由規範阻尼修正表，依 }\xi_{eD}\text{ 查得）}$$

| 等效阻尼比 $\xi_{eD}$ | $B_1$（長週期，參考值） |
|----------------------|----------------------|
| ≤ 0.02 | 0.8 |
| 0.05 | 1.0 |
| 0.10 | 1.2 |
| 0.20 | 1.5 |
| 0.30 | 1.7 |
| ≥ 0.50 | 2.0 |

---

### Step 7：計算新的設計位移 $D_D^{(\text{new})}$

由提示公式（推導自等效 SDOF 的反應譜位移）：

$$\boxed{D_D^{(\text{new})} = \frac{g}{4\pi^2}\cdot\frac{S_{aD}(T_{eD})\cdot T_{eD}^2}{B_1}}$$

其中 $S_{aD}(T_{eD})$ 為設計地震反應譜在等效週期 $T_{eD}$ 對應的 5% 阻尼加速度係數。

> **推導說明：** 等效線性 SDOF 系統的譜位移為
> $S_D(T_{eD}) = \dfrac{g \cdot S_{aD}(T_{eD}) \cdot T_{eD}^2}{4\pi^2}$，
> 除以阻尼修正係數 $B_1$（修正為實際阻尼 $\xi_{eD}$ 對應的位移）即得 $D_D$。

---

### Step 8：收斂判斷

比較新舊設計位移：

$$\text{若}\quad \left|\frac{D_D^{(\text{new})} - D_D}{D_D}\right| \leq 5\% \quad\Rightarrow\quad \text{收斂，接受 }D_D^{(\text{new})}$$

$$\text{否則，令}\quad D_D \leftarrow D_D^{(\text{new})}\quad\text{，回到 Step 2 重新迭代}$$

---

### 【完整迭代流程圖】

```
輸入：W, n, k_o, F_y, α, S_aD(T), B_1 table
         ↓
初始假設 D_D⁽⁰⁾（如 D_y 的數倍）
         ↓
 ┌─ Step 2: K_eD = n[F_y(1-α)/D_D + αk_o]
 │  Step 3: T_eD = 2π√(W/K_eD·g)
 │  Step 4: A_TD = 4nF_y(1-α)(D_D - F_y/k_o)
 │  Step 5: ξ_eD = A_TD/(2π·K_eD·D_D²)
 │  Step 6: B₁ ← 查表(ξ_eD)
 │  Step 7: D_D_new = (g/4π²)·S_aD(T_eD)·T_eD²/B₁
 │         ↓
 │  |D_D_new - D_D|/D_D ≤ 5%？
 │         ↓ 否
 └─ D_D ← D_D_new
         ↓ 是
   輸出：D_D（設計位移）
         K_eD（等效勁度）
         T_eD（等效週期）
         ξ_eD（等效阻尼比）
```

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 5.1 為何用割線勁度而非初始勁度？

初始勁度 $k_o$ 描述未降伏前的剛性，但在設計地震下隔震器已降伏（$D_D \gg D_y$）。使用割線勁度（secant stiffness）才能正確反映在最大位移附近的平均剛性，使等效線性系統的最大位移與真實非線性系統接近。

### 5.2 等效阻尼比的來源：能量法

$$\xi_{eD} = \frac{E_D}{4\pi E_S}$$

其中：
- $E_D = A_{TD}$：每個週期消散的能量（遲滯迴圈面積）
- $E_S = \frac{1}{2} K_{eD} D_D^2$：最大彈性應變能

代入得 $\xi_{eD} = \frac{A_{TD}}{2\pi K_{eD} D_D^2}$，與題目提示公式完全一致。

### 5.3 阻尼修正係數 $B$ 的物理意義

$B_1$ 的作用：把 5% 阻尼的反應譜位移換算成 $\xi_{eD}$（通常 15%~30%）阻尼的實際位移。阻尼越大，位移越小，故 $B_1 > 1$ 時 $D_D$ 會降低。

$$D_D = \frac{S_D^\text{5\%}(T_{eD})}{B_1}$$
