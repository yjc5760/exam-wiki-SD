# 考題編號：SD-2005-3

**主分類：** `SD-U2` 耐震設計規範
**副分類：** `SD-U2-2` 建築耐震設計規範
**分析方法：** 等效靜力法（地震力分配）
**標籤：** `等效靜力法` `地震力分配` `樓層剪力` `Fx分配` `高度分布` `RC建築`

---

## 1. 原始題目重述 (Problem Restatement)

有一棟 **12 層樓鋼筋混凝土建築物**：
- **樓高：** 一樓 4 m；其餘 11 層（二至十二樓）均為 3.2 m
- **樓重：** 二樓 400 噸；三至十二樓（標準層）各 300 噸；屋頂層 200 噸；**另有屋突**
- **基底總剪力：** $V = 450$ 噸（已知）

(一) 請計算施加於各樓層之地震力 $F_i$
(二) 請計算各樓層所承受之剪力 $V_i$

> 本題為純文字條件，無附圖。

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
等效靜力法（Equivalent Static Force Method）中，基底剪力依「重量×高度」比例分配至各樓層：

$$F_i = \frac{W_i \cdot h_i}{\displaystyle\sum_{j} W_j h_j} \cdot V$$

物理意義：高樓層質量受到地震時動態放大效應較大（慣性力隨高度增加而增大），因此分配較大比例的地震力。

**出題者意圖：**
- 測驗高度分布法（Fx 分配法）的應用
- 測驗 12 層建築的樓層高度計算（含不同首層高度）
- 測驗樓層剪力的累積計算（從頂部往下逐層加總）
- 測驗屋突（penthouse）的規範處理判斷

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

### 作戰計畫
1. 整理各樓層高度 $h_i$（從地面量起）
2. 計算各樓層 $W_i h_i$，求總和 $\sum W_j h_j$
3. 依比例分配得 $F_i = (W_i h_i / \sum W_j h_j) \times V$
4. 從頂部累加得各樓層剪力 $V_i$

### 關鍵陷阱

| 陷阱 | 說明 | 應對 |
|------|------|------|
| ① 高度起算錯誤 | 各樓層高度 $h_i$ 應從**地面（基礎頂）**量起，非層高 | 二樓 $h_2 = 4.0$ m；三樓 $h_3 = 4.0+3.2 = 7.2$ m |
| ② 屋頂層忘計入 | 屋頂層（12F以上）有 200 噸，應列入計算 | 屋頂高度 = $4.0 + 11 \times 3.2 = 39.2$ m |
| ③ 屋突處理不當 | 屋突重量若 ≤ 頂層重的 25% 可合併頂層計算 | 本題未另給屋突重量，視為已含於 200 噸中 |
| ④ 一樓力為零 | 一樓地面層 $h_1 = 0$，故 $F_1 = 0$（不需列） | 分配公式只對有高度的樓層有意義 |
| ⑤ 剪力方向累加 | 樓層剪力需從**頂部往下累加**（不是由下往上） | $V_i = \sum_{j \geq i} F_j$（含本樓層） |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
求各樓層地震力 $F_i$ 與各樓層剪力 $V_i$（共 12 個樓層高度 + 屋頂，13 組值）。

### 本題關鍵公式（依計算順序）

$$h_i = h_{\rm 1F} + (i-2) \cdot h_{\rm std} \quad (i \geq 2)$$

$$h_{\rm roof} = h_{\rm 1F} + n_{\rm std} \cdot h_{\rm std}$$

$$F_i = \frac{W_i h_i}{\displaystyle\sum_j W_j h_j} \cdot V$$

$$V_i = \sum_{j=i}^{\rm roof} F_j \quad \text{（樓層剪力由頂部累加）}$$

### L1：題目直接給定

| 符號 | 數值 | 說明 |
|------|------|------|
| $h_{\rm 1F}$ | 4.0 m | 一樓層高 |
| $h_{\rm std}$ | 3.2 m | 標準層層高（2F~12F 及屋頂） |
| $W_2$ | 400 t | 二樓重量 |
| $W_{3\sim12}$ | 300 t（各） | 三至十二樓標準層重 |
| $W_{\rm roof}$ | 200 t | 屋頂層重（含屋突） |
| $V$ | 450 t | 基底總剪力（已知） |

### L2：需知識點推導

**高度計算**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|
| $h_i$ | $4.0 + (i-2) \times 3.2$ m，$i \geq 2$ | |
| $h_{\rm roof}$ | $4.0 + 11 \times 3.2 = 39.2$ m | |
| $\sum W_j h_j$ | 逐層乘積累加 | |

**地震力分配**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|
| $F_i$ | $(W_i h_i / \sum W_j h_j) \times V$ | |
| $V_i$ | $\sum_{j \geq i} F_j$（從頂累加） | |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|-------|
| 高度分布法物理基礎 | 地震慣性力 ∝ 加速度 × 質量；高樓層加速度放大 → 高度線性分布假設 | |
| 屋突規範處理 | 舊台灣規範：屋突重量 ≤ 頂層重 25% 可合併；> 25% 另計一層 | |
| 老版 $F_t$ 項 | 部分版本含頂層集中力 $F_t = 0.07TV$（$T > 0.7$ s 時）；本題直接給 V，故 $F_t = 0$ | |

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**1. 屋突（Penthouse）的處理**

依 2005 年適用的台灣建築耐震設計規範：
- 屋突重量 ≤ 頂層重的 **25%**：合併頂層重量計算，以屋頂層高度 $h_{\rm roof}$ 分配地震力
- 屋突重量 > 25%：另計一層（增加第 13 個力）

本題屋突重量未給出 → 視為已包含於屋頂層 200 噸中（或視為 ≤ 25% 門檻），直接計入 $W_{\rm roof} = 200$ t。

**2. 頂層集中力 $F_t$（舊版規範）**

部分舊版規範（如 ATC-3、早期台灣規範）在週期較長（$T > 0.7$ s）時，需於頂層再加一個集中力：
$$F_t = 0.07TV \leq 0.25V$$

本題題目直接給定基底剪力 $V = 450$ t，未提供週期 $T$，因此適用簡化分配（等同 $F_t = 0$）。若實際 $T > 0.7$ s，需重新分配。

**3. 一樓是否有地震力**

本題一樓（$h_1 = 0$）不分配地震力（$F_1 = W_1 \times 0 / \sum W_j h_j = 0$）。即便一樓有重量，由於高度為零，貢獻為零。一樓重量影響的是總樓重 $W$（用於計算基底剪力 $V$），但本題 $V$ 已知，故不影響計算。

**4. 考場建議**

- 先整理一張高度表再乘以重量，避免 $h_i$ 計算出錯
- $\sum W_i h_i = 74,240$ t·m 是全題的關鍵數字，建議明確寫出並驗算
- 樓層剪力表格從頂往下填，可利用「逐層累加」迅速完成，且最後應等於 $V = 450$ t 作為自我驗算
