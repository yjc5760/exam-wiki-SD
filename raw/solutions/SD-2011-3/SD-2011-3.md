## 題目敘述

一升降電梯系統可理想化為一**懸吊彈簧質塊系統**：
- 電梯質量：m
- 彈簧彈性係數：k
- 今彈簧之**上端以等速度 v 上升**
- 以電梯之**上升距離 u** 為自由度

試求：
(一) 建立該動態系統之運動方程式（10分）
(二) 若該電梯之起始狀態為靜止，求 u(t)（15分）

---

## 解題步驟

### 前置：系統定義與靜平衡

**座標系定義（向上為正）：**
- u：電梯質塊自初始靜止位置向上的位移（即「上升距離」，u(0) = 0）
- 彈簧上端位置：初始為 y₀，之後以等速 v 上升 → 上端相對初始位置的位移 = v·t

**靜平衡條件：**

初始（t = 0）電梯靜止，彈簧已有靜態伸長量：

$$\delta_{st} = \frac{mg}{k}$$

此靜態伸長量使彈簧拉力 = 重力 = mg，系統平衡。

---

### 步驟一：（一）運動方程式

**t 時刻系統幾何關係：**

- 彈簧上端比初始位置高出：v·t
- 電梯比初始位置高出：u
- 彈簧伸長量 = 靜態伸長量 + (上端上升量 − 電梯上升量) = δ_st + (v·t − u)

**電梯垂直方向 Newton 第二定律（向上為正）：**

$$m\ddot{u} = \underbrace{k[\delta_{st} + (vt - u)]}_{\text{彈力（向上）}} - \underbrace{mg}_{\text{重力（向下）}}$$

$$m\ddot{u} = k\delta_{st} + k(vt - u) - mg$$

由靜平衡 $k\delta_{st} = mg$，消去靜態項：

$$m\ddot{u} = k(vt - u)$$

整理得**運動方程式**：

$$\boxed{m\ddot{u} + ku = kvt}$$

或等價形式：

$$\ddot{u} + \omega_0^2 u = \omega_0^2 vt, \quad \omega_0 = \sqrt{\frac{k}{m}}$$

> **物理意義：** 等速上升的彈簧上端對電梯施加一個線性增長（Ramp）的等效激振力 kvt。這與地震中「斜坡型地面運動」的數學形式完全相同。

---

### 步驟二：（二）求 u(t)，初始靜止條件

**初始條件：** u(0) = 0，u̇(0) = 0

求解 $\ddot{u} + \omega_0^2 u = \omega_0^2 vt$（無阻尼 SDOF，Ramp 型激振）

**1. 特解（Particular solution）**

設 $u_p = At + B$，代入方程式：

$$\ddot{u}_p + \omega_0^2 u_p = 0 + \omega_0^2(At + B) = \omega_0^2 vt$$

比較係數：
- t 的係數：$\omega_0^2 A = \omega_0^2 v \implies A = v$
- 常數項：$\omega_0^2 B = 0 \implies B = 0$

$$u_p = vt$$

**2. 齊次解（Homogeneous solution）**

$$u_h = C_1\cos(\omega_0 t) + C_2\sin(\omega_0 t)$$

**3. 全解**

$$u(t) = C_1\cos(\omega_0 t) + C_2\sin(\omega_0 t) + vt$$

**4. 代入初始條件**

初始位移 u(0) = 0：

$$C_1 \cdot 1 + C_2 \cdot 0 + 0 = 0 \implies C_1 = 0$$

初始速度 $\dot{u}(0) = 0$（注意 $\dot{u}_h = -C_1\omega_0\sin + C_2\omega_0\cos$）：

$$\dot{u}(0) = C_2\omega_0 + v = 0 \implies C_2 = -\frac{v}{\omega_0}$$

**最終解：**

$$\boxed{u(t) = vt - \frac{v}{\omega_0}\sin(\omega_0 t)}$$

其中 $\omega_0 = \sqrt{k/m}$，可改寫為：

$$u(t) = v\left[t - \frac{\sin(\omega_0 t)}{\omega_0}\right]$$

---

## 驗算

**驗算初始條件：**

$$u(0) = v[0 - 0] = 0 \checkmark$$

$$\dot{u}(t) = v[1 - \cos(\omega_0 t)]$$

$$\dot{u}(0) = v[1 - 1] = 0 \checkmark$$

**驗算方程式（代入 u(t)）：**

$$\ddot{u}(t) = v\omega_0\sin(\omega_0 t)$$

$$\ddot{u} + \omega_0^2 u = v\omega_0\sin(\omega_0 t) + \omega_0^2\left[vt - \frac{v}{\omega_0}\sin(\omega_0 t)\right]$$

$$= v\omega_0\sin(\omega_0 t) + \omega_0^2 vt - v\omega_0\sin(\omega_0 t) = \omega_0^2 vt \checkmark$$

---

## 計算彙整

| 項目 | 結果 |
|-----|------|
| 運動方程式 | mü + ku = kvt |
| 自然頻率 | ω₀ = √(k/m) |
| 特解 | u_p = vt |
| 全解（零初始條件） | **u(t) = vt − (v/ω₀)sin(ω₀t)** |
| 速度 | u̇(t) = v[1 − cos(ω₀t)] |
| 加速度 | ü(t) = vω₀sin(ω₀t) |

## 解題關鍵觀念

### 1. 靜平衡消去技巧

懸吊系統以「相對於靜平衡位置的位移」或「自靜止位置的上升距離」為自由度時，**靜態重力與靜態彈力相互抵消**，運動方程式只剩動態部分。這是結構動力學最重要的技巧之一。

### 2. Ramp（線性增長）激振的特解形式

| 激振形式 | f(t) | 特解形式 |
|---------|------|---------|
| 常數力 | F₀ | A（常數） |
| **線性增長（Ramp）** | **F₀t** | **At + B** |
| 諧和激振 | F₀sin(ωt) | Asin(ωt) + Bcos(ωt) |

Ramp 激振的特解為 vt，代表電梯在平均上「跟隨」彈簧上端的等速運動。

### 3. 物理解釋

$$u(t) = \underbrace{vt}_{\text{穩態跟隨}} - \underbrace{\frac{v}{\omega_0}\sin(\omega_0 t)}_{\text{自由振盪（因突然起動引發）}}$$

- 第一項 vt：電梯「平均」以速度 v 上升，跟隨彈簧上端
- 第二項：由於起動瞬間速度從 0 突然變化（非平滑），誘發振幅 v/ω₀ 的自由振動
- 振幅 $v/\omega_0 = v\sqrt{m/k}$：彈簧愈硬（k 大）或質量愈輕，振幅愈小

電梯速度：$\dot{u}(t) = v[1 - \cos(\omega_0 t)]$，在 0 到 2v 之間振盪，平均速度等於 v。

### 4. 與地震工程的連結

本題的「等速基礎運動」在地震工程中對應「速度脈衝型地面運動」（Velocity Pulse），是近斷層地震（Near-fault earthquake）的特徵。分析方法完全相同：基礎位移 → 等效激振力 → 求解 SDOF 響應。

### 5. 常見陷阱

- ❌ 忘記靜態抵消：直接寫 mü + ku = kvt + mg（重複計入重力）
- ❌ 特解猜 u_p = Ct²（適用於常數激振力，不適用 Ramp 型）
- ❌ 初速度條件：u̇(0) = 0 代入的是全解的速度，含特解項 v，故 C₂ = −v/ω₀ 不是零
