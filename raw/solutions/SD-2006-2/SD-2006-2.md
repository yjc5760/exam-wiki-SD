## 1. 原始題目重述 (Problem Restatement)

- 各層樓質量：$m_1 = m_2 = m = 2000\ \text{kg}$
- 各層側向勁度：$k_1 = k_2 = k = 4\times10^6\ \text{N/m}$

**子題(一)（6分）：** 求各振態週期。

**子題(二)（6分）：** 求各振態有效質量佔總質量之百分比。

**子題(三)（10分）：** 結構承受基底振動，動力分析所得：
- 第一振態等效樓層橫力：$\{F\}^{(1)} = \{11358,\ 18377\}^T\ \text{N}$（下層、上層）
- 第二振態等效樓層橫力：$\{F\}^{(2)} = \{3997,\ -2470\}^T\ \text{N}$

以 SRSS 法則組合，繪出樓層剪力圖並標示數值。

**子題(四)（5分）：** 在相同基底振動下，改以耐震規範靜力分析法之樓層橫力豎向分配法則，重新繪出樓層剪力圖並標示數值（不考慮頂層外加集中橫力）。

**子題(五)（6分）：** 若非剪力屋架，尚需考慮何種變數以計算勁度矩陣？此時樓層剪力將變大或變小？為什麼？

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**步驟順序（子題一、二）：**
1. 建立 $[M]$ 和 $[K]$（剪力屋架勁度矩陣）
2. 特徵值方程式求 $\omega_1^2, \omega_2^2$
3. 代回求振態 $\{\phi\}^{(1)}, \{\phi\}^{(2)}$
4. 計算各振態有效質量

**步驟順序（子題三）：**
5. 各振態樓層力 → 各振態樓層剪力
6. SRSS 組合（在剪力上做 SRSS）
7. 繪剪力圖

**步驟順序（子題四）：**
8. 以 SRSS 基底剪力為 $V_{base}$，按質量×高度比例分配
9. 求靜力法各樓層橫力 → 靜力法樓層剪力

**關鍵陷阱：**

1. **陷阱①：剪力屋架勁度矩陣**
   第一層勁度 $k_1$ 同時承擔第一層位移和第二層對第一層的相對位移。
   - $K_{11} = k_1 + k_2 = 2k$（不是 $k$）
   - $K_{12} = K_{21} = -k_2 = -k$
   - $K_{22} = k_2 = k$

2. **陷阱②：SRSS 在樓層剪力做，不在樓層力做**
   直接對樓層力 SRSS 後再求剪力（錯誤），應先分別求各振態的樓層剪力，再 SRSS 組合。

3. **陷阱③：第二振態分力為負值**
   $-2470\ \text{N}$ 取 SRSS 時用**平方**（正負號自動消失），不可取絕對值後再組合（兩者結果相同，但概念要清楚）。

4. **陷阱④：靜力分配公式的分母**
   $\sum w_i h_i = m \cdot g \cdot h + m \cdot g \cdot 2h = 3mgh$，若誤用層高 $h$ 而非累積高度 $2h$，分配比例會錯。

## 3.5 變數層次分析（Variable Hierarchy Analysis）

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`依 SRSS 法則求樓層剪力圖，並與靜力分析法對比`

### 本題關鍵公式（依計算順序）

$$\text{Step 1: }[K] = \begin{bmatrix}k_1+k_2 & -k_2\\-k_2 & k_2\end{bmatrix} = \begin{bmatrix}2k & -k\\-k & k\end{bmatrix}$$

$$\text{Step 2: }\det([K]-\omega^2[M])=0 \implies \omega^4 - 3\frac{k}{m}\omega^2 + \frac{k^2}{m^2} = 0$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

> 📊 互動圖：`SD-2006-2-modal-viz.html`

### 子題(一)：各振態週期

#### 建立質量矩陣與勁度矩陣

$$[M] = \begin{bmatrix}2000 & 0\\0 & 2000\end{bmatrix}\ \text{kg}, \qquad
[K] = \begin{bmatrix}8\times10^6 & -4\times10^6\\-4\times10^6 & 4\times10^6\end{bmatrix}\ \text{N/m}$$

> 策略注解：剪力屋架中，$K_{11} = k_1+k_2$ 因為層間相對位移 $x_1$ 使兩層彈簧均受力；$K_{22} = k_2$ 僅上層彈簧受力。

#### 特徵值方程式

$$\det([K]-\omega^2[M]) = 0$$

$$\det\begin{bmatrix}8\times10^6-2000\omega^2 & -4\times10^6\\-4\times10^6 & 4\times10^6-2000\omega^2\end{bmatrix} = 0$$

展開：

$$(8\times10^6-2000\omega^2)(4\times10^6-2000\omega^2) - (-4\times10^6)^2 = 0$$

令 $u = 2000\omega^2$（單位 N/m）：

$$(8\times10^6-u)(4\times10^6-u) - 16\times10^{12} = 0$$

$$u^2 - 12\times10^6 u + 32\times10^{12} - 16\times10^{12} = 0$$

$$u^2 - 12\times10^6 u + 16\times10^{12} = 0$$

求解：

$$u = \frac{12\times10^6 \pm \sqrt{144\times10^{12}-64\times10^{12}}}{2} = \frac{12\times10^6 \pm \sqrt{80\times10^{12}}}{2} = \frac{(12\pm4\sqrt{5})\times10^6}{2} = (6\pm2\sqrt{5})\times10^6$$

換算 $\omega^2 = u/2000$：

$$\omega_1^2 = \frac{(6-2\sqrt{5})\times10^6}{2000} = (3-\sqrt{5})\times10^3 = 763.9\ \text{rad}^2/\text{s}^2$$

$$\omega_2^2 = \frac{(6+2\sqrt{5})\times10^6}{2000} = (3+\sqrt{5})\times10^3 = 5236.1\ \text{rad}^2/\text{s}^2$$

$$\omega_1 = \sqrt{763.9} = 27.64\ \text{rad/s}, \qquad \omega_2 = \sqrt{5236.1} = 72.36\ \text{rad/s}$$

$$\boxed{T_1 = \frac{2\pi}{\omega_1} = \frac{2\pi}{27.64} = 0.227\ \text{s}}$$

$$\boxed{T_2 = \frac{2\pi}{\omega_2} = \frac{2\pi}{72.36} = 0.0868\ \text{s}}$$

#### 振態向量

**第一振態** $(\omega_1^2 = 763.9\ \text{rad}^2/\text{s}^2)$：

由第一列方程（令 $\phi_{11}=1$）：

$$\phi_{21} = \frac{K_{11}-\omega_1^2 m}{K_{21}} = \frac{(8-2\times0.7639)\times10^6}{4\times10^6} = \frac{(8-1.5279)\times10^6}{4\times10^6} = \frac{6.4721}{4} = 1.618 = \frac{1+\sqrt{5}}{2}\ (黃金比例)$$

$$\{\phi\}^{(1)} = \begin{Bmatrix}1\\1.618\end{Bmatrix}$$

**第二振態** $(\omega_2^2 = 5236.1\ \text{rad}^2/\text{s}^2)$：

$$\phi_{22} = \frac{8\times10^6-2000\times5236.1}{4\times10^6} = \frac{(8-10.472)\times10^6}{4\times10^6} = \frac{-2.472\times10^6}{4\times10^6} = -0.618 = \frac{1-\sqrt{5}}{2}$$

$$\{\phi\}^{(2)} = \begin{Bmatrix}1\\-0.618\end{Bmatrix}$$

### 子題(二)：各振態有效質量百分比

有效質量公式：

$$m_i^* = \frac{L_i^2}{\tilde{M}_i}, \qquad L_i = \{\phi\}_i^T[M]\{1\}, \qquad \tilde{M}_i = \{\phi\}_i^T[M]\{\phi\}_i$$

**第一振態：**

$$L_1 = 2000\times1 + 2000\times1.618 = 2000\times2.618 = 5236\ \text{kg}$$

$$\tilde{M}_1 = 2000\times1^2 + 2000\times1.618^2 = 2000\times(1+2.618) = 2000\times3.618 = 7236\ \text{kg}$$

$$m_1^* = \frac{5236^2}{7236} = \frac{27,415,696}{7236} = 3789\ \text{kg}$$

$$\boxed{\frac{m_1^*}{M_{total}} = \frac{3789}{4000} = 94.7\%}$$

**第二振態：**

$$L_2 = 2000\times1 + 2000\times(-0.618) = 2000\times0.382 = 764\ \text{kg}$$

$$\tilde{M}_2 = 2000\times1^2 + 2000\times(-0.618)^2 = 2000\times(1+0.382) = 2000\times1.382 = 2764\ \text{kg}$$

$$m_2^* = \frac{764^2}{2764} = \frac{583,696}{2764} = 211\ \text{kg}$$

$$\boxed{\frac{m_2^*}{M_{total}} = \frac{211}{4000} = 5.3\%}$$

**驗算：** $3789 + 211 = 4000\ \text{kg} = M_{total}$ ✓；$94.7\% + 5.3\% = 100\%$ ✓

> 結論：第一振態貢獻了 94.7% 的有效質量，說明此均勻剪力屋架的地震反應幾乎由第一振態控制。

### 子題(四)：耐震規範靜力分析法（豎向分配）

**前提：** 使用相同基底剪力 $V_{base} = V_1^{\text{SRSS}} = 29{,}774\ \text{N}$；不考慮頂層外加集中橫力 $F_t$（因 $T_1 = 0.227\ \text{s} < 0.7\ \text{s}$，規範規定 $F_t = 0$）。

**豎向分配公式（耐震規範標準式）：**

$$F_i = \frac{w_i h_i}{\sum_{j=1}^{n} w_j h_j} \cdot V_{base}$$

**高度假設：** 均勻剪力屋架，設各層高均為 $h$，則：
- 第一層樓版高度（從地面量）：$h_1 = h$
- 第二層樓版高度（屋頂）：$h_2 = 2h$

質量相等（$w_1 = w_2 = m$），分母：

$$\sum w_j h_j = mgh + mg(2h) = 3mgh$$

各層分配：

$$F_1^{\text{code}} = \frac{mgh}{3mgh}\times29{,}774 = \frac{1}{3}\times29{,}774 = \boxed{9{,}925\ \text{N}}$$

$$F_2^{\text{code}} = \frac{mg(2h)}{3mgh}\times29{,}774 = \frac{2}{3}\times29{,}774 = \boxed{19{,}849\ \text{N}}$$

**靜力法樓層剪力：**

$$V_2^{\text{code}} = F_2^{\text{code}} = \boxed{19{,}849\ \text{N}}$$

$$V_1^{\text{code}} = F_1^{\text{code}} + F_2^{\text{code}} = 9{,}925 + 19{,}849 = \boxed{29{,}774\ \text{N}}\ (\text{驗算: = }V_{base}\ ✓)$$

#### 靜力法樓層剪力圖

```
          ┌─────────────────────────────────────┐
          │          第二層（屋頂層）             │
          │                                     │
          │  V₂ = 19,849 N ←───────────────── │
          ├─────────────────────────────────────┤
          │          第一層                      │
          │                                     │
          │  V₁ = 29,774 N ←────────────────── │
          └──────────────────────────────────── ┘
```

### 子題(五)：非剪力屋架的勁度矩陣變數與剪力大小

#### 尚需考慮的變數

剪力屋架假設「梁無限剛、樓版不考慮轉角自由度」，僅需每層側向勁度 $k_i$（由柱彎曲剛度決定：$k_i = \sum 12EI_c/h^3$）。

若非剪力屋架（一般矩形構架），需加入：

1. **梁的彎曲勁度** $EI_b / L_b$（梁的截面積慣性矩 $I_b$、跨度 $L_b$）
   → 梁若非剛性，節點會產生轉角，側向勁度降低
2. **柱的幾何尺寸與彈性模數** $EI_c / h$（不再能以整層勁度 $k_i$ 概括，需個別柱子的 $EI_c$、$h$）
3. **節點轉角自由度** $\theta$
   → 自由度從 $n$（樓層數）增加到 $n + $ 節點數（或用凝縮法消去，需梁柱剛度比資訊）
4. **柱的軸向剛度** $EA_c / h$（高寬比大時，柱的軸向變形影響傾覆效應，不可忽略）

#### 樓層剪力變大或變小？

$$\boxed{\text{樓層剪力變小}}$$

**原因：**
- 一般矩形構架（梁有限剛度）的側向勁度**低於**剪力屋架（梁無限剛度假設）
- 勁度降低 → 自然週期**延長**（$T \propto 1/\sqrt{k}$）
- 在反應譜的**速度控制段或位移控制段**（$T > T_s$）：譜加速度 $S_a \propto 1/T$，週期延長使 $S_a$ 降低
- 設計地震力 $V = S_a \cdot W / g$ 因此**減小**，對應各層剪力均**變小**
- 換言之：剪力屋架的剛假設高估了實際勁度，以它計算所得剪力比真實值**偏大（保守）**

> 補充：若建築週期已在平台段（短週期，$S_a = S_{DS}$），延長週期仍在平台段內則剪力不變；但對多數結構，延長週期後會落入下降段，使剪力降低。

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 爭議點 1：SRSS 的適用條件

SRSS 假設各振態反應統計上獨立（不相關）。精確條件是振態頻率相差 $>10\%$。本題：
$$r = \omega_1/\omega_2 = 27.64/72.36 = 0.382 \implies \text{頻率相差} = (1-r)/1 \approx 62\%$$
遠超 10%，SRSS 適用；若頻率相近，應改用 CQC。

### 爭議點 2：靜力法分配公式的假設

規範靜力法的 $w_i h_i$ 分配對應第一振態呈線性（倒三角）分布的近似。本題兩振態的有效質量比 94.7%：5.3%，第一振態主導，因此靜力法近似相當合理，誤差僅在頂層剪力差 7%。

### 進階補充：黃金比例出現的數學根源

本題振態比 $\phi_{21}/\phi_{11} = (1+\sqrt{5})/2 = \varphi \approx 1.618$ 正是黃金比例，源於等質量等勁度 2-DOF 系統的特徵值方程式 $u^2-12\times10^6 u+16\times10^{12}=0$ 的無因次化結果。這是結構動力學中少數可精確解析的系統之一。
