## §2 (一) 基礎隔震基本概念（3 分）

### 核心概念

基礎隔震（base isolation）是在結構物底部與基礎之間設置具有**低水平勁度、高鉛直勁度**的隔震裝置，其目的在於：

1. **延長結構自然週期**：隔震系統使整體結構週期大幅延長（通常至 2–4 s），遠離地震波之主要能量頻段（0.1–1.0 s）。

2. **避免共振、降低地震傳遞力**：週期拉長後，結構所承受之譜加速度 $S_a$ 顯著下降，傳遞至上部結構的地震力（$F=m\cdot S_a$）大幅減少。

3. **集中變形於隔震層**：上部結構幾乎作剛體運動，層間變形（及破壞）集中於隔震層，而非分散於各樓層。

4. **隔震層附加消能**：通常輔以阻尼（液態阻尼器、摩擦耗能、LRB 鉛芯等）以限制隔震位移。

### 概念示意

```
       上部結構（幾乎剛體運動）
      ┌─────────────────┐
      │                 │
      ├─────────────────┤  ← 隔震層（低 k_H、大阻尼）
      │  隔震裝置        │
      └────────/////////┘  ← 基礎（隨地面運動）
```

**關鍵原理：** 上部結構週期 $T_{isolated} = 2\pi\sqrt{m/k_H} \gg T_{fixed-base}$，對應之 $S_a$ 遠低於短週期端，上部結構所受加速度與地震力因此大幅降低。

## §3 (二) 摩擦型隔震系統原理（3 分）

### 主要原理

摩擦型隔震系統（sliding isolation system）利用**滑動界面的摩擦力**達到以下效果：

1. **力的限制（force limiting）**：當水平地震力超過靜摩擦力閾值 $F_{max} = \mu W$（$\mu$：摩擦係數，$W$：上部結構重量），滑動面開始相對滑動，最大傳遞水平力被限制在 $\mu W$，不再隨地震強度增加。

2. **週期轉換**：對於平面滑動系統（flat sliding），一旦滑動，上部結構在水平方向失去彈性約束，自然週期趨近無限大（理想狀況），對高頻地震波不感應。

3. **耗能（energy dissipation）**：每個滑動循環中，摩擦力做功消耗地震能量，產生等效阻尼，限制滑動位移。

4. **缺點（純平面型）**：滑動結束後無復位力，結構停止時可能存在永久位移。→ 改良型：採用球面（FPS）以提供復位力。

**傳遞力上限：**
$$F_{max} = \mu W$$

## §4 (三) 摩擦單擺隔震系統振動週期推導（6 分）

### 系統描述

摩擦單擺隔震系統（Friction Pendulum System，FPS）在凹形球面（曲率半徑 $r$）上設置滑動面，如下圖：

```
             r（曲率半徑）
            ◯ ← 球面中心
           /|
          / |r
         /  |
        ◆ ← 滑塊（質量 m = W/g）
       /θ\
      /    \
     /  x   \  ← 水平位移
    ↙         
凹形球面
```

### 力學分析（忽略摩擦力）

設滑塊在球面上偏移角度 $\theta$，對應水平位移 $x$，則：

$$\sin\theta = \frac{x}{r} \quad (\text{精確})$$

小角度近似（$\theta$ 小時，$\sin\theta \approx \tan\theta \approx \theta$）：

$$\theta \approx \frac{x}{r}$$

**對滑塊進行受力分析（忽略摩擦力）：**

- 重力：$W = mg$（鉛直向下）
- 法向力：$N$（垂直於球面，指向球心）
- 水平方向：法向力水平分量 = 復位力

鉛直方向平衡：
$$N\cos\theta = W \implies N = \frac{W}{\cos\theta}$$

水平方向（Newton 第二定律）：
$$m\ddot{x} = -N\sin\theta = -W\frac{\sin\theta}{\cos\theta} = -W\tan\theta$$

小角度近似 $\tan\theta \approx \sin\theta \approx x/r$：

$$m\ddot{x} = -W\cdot\frac{x}{r} = -\frac{mg}{r}\cdot x$$

即：

$$\ddot{x} + \frac{g}{r}\,x = 0$$

### 振動週期

此為簡諧運動（SHM），自然頻率：

$$\omega^2 = \frac{g}{r} \implies \omega = \sqrt{\frac{g}{r}}$$

$$\boxed{T_{FPS} = 2\pi\sqrt{\frac{r}{g}}}$$

### 類比單擺

FPS 的振動週期與等效擺長為 $r$ 的單擺完全相同（故稱「摩擦**單擺**隔震系統」）。週期**僅與 $r$ 和 $g$ 有關，與上部結構質量（重量）無關**——這是 FPS 的重要特性：不同重量的結構置於同一球面上，週期相同。

## §5 (四) 側向勁度（3 分）

由 §4 推導，水平復位力為：

$$F = -\frac{W}{r}\cdot x = -k_{eff}\cdot x$$

因此等效側向勁度（effective lateral stiffness）：

$$\boxed{k_{eff} = \frac{W}{r}}$$

**驗算：**

$$T = 2\pi\sqrt{\frac{m}{k_{eff}}} = 2\pi\sqrt{\frac{W/g}{W/r}} = 2\pi\sqrt{\frac{r}{g}} \quad \checkmark$$

**物理意義：** 勁度 $k_{eff} = W/r$ 隨重量 $W$ 增加而增大，但週期 $T = 2\pi\sqrt{r/g}$ 不變，因為較重的結構雖然勁度較大，其慣性（質量 $m = W/g$）也等比例增加，兩者抵銷。

## §6 計算重點與補充說明

### §6.1 答案彙整

| 子題 | 答案 |
|------|------|
| (一) 基本概念 | 延長週期 + 力的衰減 + 集中變形於隔震層 |
| (二) 摩擦型原理 | 利用 $F \leq \mu W$ 限制傳力；滑動耗能 |
| (三) FPS 週期 | $T = 2\pi\sqrt{r/g}$ |
| (四) 側向勁度 | $k_{eff} = W/r$ |

### §6.2 FPS 與 LRB 比較

| 特性 | FPS（摩擦單擺）| LRB（鉛芯橡膠）|
|------|--------------|--------------|
| 週期控制 | 由 $r$ 決定，與 $W$ 無關 | 由橡膠層厚度與面積決定 |
| 復位機制 | 球面幾何提供重力復位 | 橡膠彈性提供復位 |
| 耗能機制 | 滑動摩擦力 | 鉛芯塑性變形 |
| 主要優點 | 承載能力強、可承受巨大重量 | 工廠製造、品質均一 |

### §6.3 常見錯誤

| 錯誤 | 正確觀念 |
|------|---------|
| 以為 FPS 週期與質量有關 | $T = 2\pi\sqrt{r/g}$，週期與質量無關 |
| 摩擦力在推導中未說明「忽略」| 題目明確說忽略摩擦力影響 |
| $k_{eff}$ 以 $mg/r$ 表示（g混入）| 應用 $W$ 表示：$k = W/r$（與上下一致）|
| 忘記說明復位力與球面幾何的關係 | FPS 靠球面曲率提供重力分量作為復位力 |

### §6.4 設計意涵

- **選擇曲率半徑 $r$：** 設計者可藉由調整 $r$ 來設定隔震週期，例如欲達 $T=3$ s，需 $r = g(T/2\pi)^2 = 9.81 \times (3/2\pi)^2 \approx 2.24$ m。
- **位移需求：** 球面半徑 $r$ 越大，允許的水平位移也越大，但隔震台尺寸需對應增加。
- **摩擦力影響（題目未要求但值得了解）：** 實際上摩擦會增加等效阻尼 $\xi_{eq}$，修正週期略短，且傳遞力為 $F = k_{eff} x + \mu W \cdot \text{sgn}(\dot{x})$。
