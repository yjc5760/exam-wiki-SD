# 阻尼比（ξ）（Damping Ratio）

**概念 ID：** SD-C-003
**知識分類：** `SD-U1-1` 結構動力基本性質及原理

## 定義

實際阻尼係數與臨界阻尼係數之比。欠阻尼（ξ<1）為工程結構典型狀態，典型值 2%~5%。

## 關鍵公式

$$
ξ = c / (2mω₀) = c / ccr；ccr = 2mω₀；ωd = ω₀√(1−ξ²)
$$
