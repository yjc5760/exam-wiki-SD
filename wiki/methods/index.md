# 解題方法論（SD）

> 本目錄收錄結構動力分析與耐震設計的解題方法頁，由 Cowork 於解題時生成。

| 方法 | 適用題型 |
|------|---------|
| 模態疊加法 | MDOF 多自由度系統動力分析 |
| Rayleigh-Ritz 法 | 廣義自由度法、基本頻率估算 |
| 等效靜力法 | 建築耐震設計規範、橋梁 α 法 |
| Duhamel 積分 | 任意激振 SDOF 時域反應 |
| SRSS / CQC 組合 | 反應譜多模態疊加 |
| 等效線性化 | 隔震/消能系統非線性元件 |

*（方法頁由 compile-all 或 Cowork 解題時自動建立）*
