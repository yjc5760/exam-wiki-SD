# 共振破壞（Resonance Failure）

## 定義

激振頻率接近結構自然頻率（r ≈ 1）時，動力放大係數急遽上升，位移趨近靜位移的 **1/(2ξ)** 倍。若阻尼極小（ξ < 1%），可達靜位移數十至數百倍，造成結構破壞。

## 觸發條件

| 條件 | 說明 |
|------|------|
| 頻率比 r = Ω/ω₀ ≈ 1 | 機械振動最常見；地震中若結構週期 ≈ 地盤卓越週期同樣危險 |
| 阻尼比 ξ < 5% | 鋼結構（1–2%）比 RC（3–5%）更敏感 |
| 激振持續時間長 | 穩態共振需時間積累；短暫脈衝不足以達到穩態 |

## 動力放大係數（D）

$$D = \frac{1}{\sqrt{(1-r^2)^2 + (2\xi r)^2}}$$

共振時（r = 1）：

$$D_{max} = \frac{1}{2\xi}$$

## 出現題型

| 題號 | 考點 |
|------|------|
| SD-U1-3 類題 | 頻率比計算、共振時最大位移 |
| SD-U2-2 類題 | 結構週期與地盤卓越週期接近的危險性 |

## 防治策略

- **調頻**：改變結構勁度或質量，使 r 遠離 1
- **增阻**：附加阻尼器（粘滯阻尼器、TMD）
- **隔震**：延長 T 遠離地盤卓越週期
