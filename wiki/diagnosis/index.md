# 題型診斷層（SD）

> 拿到考題時，依此決策樹快速判斷解題路徑。

## 診斷流程

```
讀題 → 識別自由度數目
  ├─ 1個自由度 (SDOF)
  │    ├─ 無外力 → 自由振動：求 ω₀、T、對數衰減率
  │    ├─ 諧和外力 → 穩態反應：動力放大係數 D、共振條件
  │    └─ 任意外力 → Duhamel 積分 / 時程分析
  │
  ├─ 多個自由度 (MDOF)
  │    ├─ 求頻率/振態 → 特徵值問題：det([K]-ω²[M])=0
  │    ├─ 地震反應 → 模態疊加法 + 反應譜 + SRSS/CQC
  │    └─ 特殊模型 → SSI（柔性基礎）、Rayleigh-Ritz（連續體）
  │
  └─ 規範設計題
       ├─ 建築耐震 → 等效靜力法：V = Sa·W/g
       ├─ 橋梁耐震 → α 法：P = A·S·C·W；Rayleigh 週期
       └─ 隔震/消能 → 等效線性化：Keff、βeff、DD
```

## 關鍵判斷問題

1. **題目給了什麼？** m、k、c → SDOF/MDOF；W、T → 規範題
2. **求什麼？** 頻率→特徵值；位移→反應譜/時程；設計力→等效靜力
3. **有無特殊效應？** 軸壓（幾何勁度）、SSI（柔性基礎）、非線性（隔震）
