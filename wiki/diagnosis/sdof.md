# SDOF 題型診斷

> 適用：`SD-U1-1`、`SD-U1-2`、`SD-U1-3`（單自由度系統）

## 快速識別

| 題目特徵 | 判斷 |
|---------|------|
| 給 m、k、c（或 ξ）三個量 | 典型 SDOF |
| 只問「自然頻率」、「週期」 | 自由振動 |
| 給 F(t) = F₀sin(Ωt) | 穩態強迫振動 |
| 給加速度時程 üg(t) | 地震激振 → Duhamel 積分 |
| 給「對數衰減率」或「連續峰值比」 | 自由衰減 → 求 ξ |

---

## 決策樹

```
SDOF 題
  │
  ├─ 無外力（自由振動）
  │    ├─ 無阻尼：ω₀ = √(k/m)，T = 2π/ω₀
  │    └─ 有阻尼：ωd = ω₀√(1-ξ²)，x(t) = e^(-ξω₀t)[A cos ωd t + B sin ωd t]
  │         └─ 給峰值比 → δ = ln(x₁/x₂) = 2πξ/√(1-ξ²)
  │
  ├─ 諧和外力 F₀sin(Ωt)（穩態）
  │    ├─ 求頻率比 r = Ω/ω₀
  │    ├─ D = 1/√[(1-r²)²+(2ξr)²]
  │    ├─ X = D·F₀/k
  │    └─ 共振 r≈1：D_max ≈ 1/(2ξ)
  │
  └─ 任意激振
       ├─ 步階力：Heaviside 解
       ├─ 脈衝：x(t) = F̂/(mωd)·e^(-ξω₀t)·sin(ωd t)
       └─ 任意 F(τ)：Duhamel 積分
```

## 常犯錯誤

| 錯誤 | 正確作法 |
|------|---------|
| r = ω₀/Ω（分子分母倒置） | r = Ω/ω₀（激振頻率 / 自然頻率）|
| 共振時 D→∞（忽略阻尼） | 共振時 D = 1/(2ξ)，阻尼決定峰值 |
| 用 ω₀ 代替 ωd 計算衰減振動 | 有阻尼自由振動頻率為 ωd = ω₀√(1-ξ²) |

## 出現頻率高的題型

- 對數衰減率求阻尼比（幾乎每 2–3 年出現一次）
- 動力放大係數與共振條件
- Duhamel 積分（步階力、矩形脈衝）
