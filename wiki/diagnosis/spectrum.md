# 反應譜分析題型診斷

> 適用：`SD-U1-3`（SDOF/MDOF 反應譜）、`SD-U2-1`（地震力設計規範）

## 快速識別

| 題目特徵 | 判斷 |
|---------|------|
| 給「設計反應譜」圖或 SDS、SD1 值 | 反應譜設計題 |
| 問「最大位移」、「最大加速度」 | 從反應譜讀 Sd 或 Sa |
| 給多個模態 + 問組合後最大反應 | SRSS 或 CQC |
| 頻率相差 > 10%，無密集模態 | 用 SRSS |
| 頻率相近（密集模態）| 用 CQC |

---

## 決策樹

```
反應譜題
  │
  ├─ SDOF 反應
  │    ├─ 已知 T → 從反應譜讀 Sd(T) 或 Sa(T)
  │    ├─ Sv ≈ (2π/T)·Sd
  │    └─ Sa ≈ (2π/T)²·Sd = ω₀²·Sd
  │
  └─ MDOF 多模態疊加
       ├─ 各模態：計算 Γᵢ → 各模態最大反應 rᵢ = Γᵢ·Sd(Tᵢ)·{φ}ᵢ
       ├─ SRSS：R = √(Σrᵢ²)
       └─ CQC：R = √(ΣΣρᵢⱼ rᵢ rⱼ)
```

## 台灣設計反應譜公式

| 區段 | 公式 | 條件 |
|------|------|------|
| 上升段 | Sa = SDS(0.4 + 0.6T/T₀) | T < T₀ |
| 平台段 | Sa = SDS | T₀ ≤ T ≤ Ts |
| 下降段 | Sa = SD1/T | T > Ts |
| 長週期段 | Sa = SD1·TL/T² | T > TL |

其中 T₀ = 0.2Ts，Ts = SD1/SDS

## 常犯錯誤

| 錯誤 | 正確作法 |
|------|---------|
| SRSS 直接加絕對值 | SRSS：√(r₁²+r₂²)，非 r₁+r₂ |
| 密集頻率仍用 SRSS | 頻率差 < 10% 需用 CQC |
| 把 Sd 當 Sa | Sd = 位移譜，Sa = 加速度譜，差 ω₀² 倍 |
