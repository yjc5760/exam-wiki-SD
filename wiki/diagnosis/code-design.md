# 規範設計題型診斷

> 適用：`SD-U2-2`（建築耐震）、`SD-U2-3`（橋梁耐震）、`SD-U3-1/2`（耐震/隔震設計）

## 快速識別

| 題目特徵 | 判斷 |
|---------|------|
| 給 W（重量）、T（週期）、SDS/SD1 | 建築等效靜力法 |
| 給 A、S、T，問橋梁地震力 | 橋梁 α 法 |
| 給隔震墊剛度、阻尼，問設計位移 | 隔震設計 |
| 問「強柱弱梁」、「容量設計」 | 耐震設計概念題 |

---

## 決策樹

```
規範設計題
  │
  ├─ 建築耐震（SD-U2-2）
  │    ├─ 基底剪力 V = Sa(T)·W/g，V ≥ SD1·Ie·W/(T·R)
  │    ├─ 樓層剪力分布：Fx = CvxV，Cvx = wₓhₓᵏ/Σwᵢhᵢᵏ
  │    └─ 層間變位角 δ = Cd·δe/Ie ≤ 容許值
  │
  ├─ 橋梁耐震（SD-U2-3，α 法）
  │    ├─ P = A·S·C·W
  │    ├─ C = 1.2S/T^(2/3) ≤ 2.5A（週期 T 需算）
  │    ├─ 週期：T = 2.01√(γ/β)（Rayleigh 法）
  │    └─ 重要性因子 I：生命線 1.5 / 主要 1.25 / 一般 1.0
  │
  └─ 隔震設計（SD-U3-2）
       ├─ TIS = 2π√(W/g·Keff)
       ├─ DD = g·SD1·TIS/(4π²·BD)
       └─ 等效阻尼比 βeff = WD/(2π·Keff·DD²)
```

## 常犯錯誤

| 錯誤 | 正確作法 |
|------|---------|
| 橋梁 γ、β 計算錯誤 | γ = ∫w·u²dx，β = ∫w·u dx（u 為靜力位移，非振態）|
| 樓層剪力分布忽略頂部集中力 | T > 0.7s 時，Ft = 0.07TV，其餘樓層用 Cvx×(V-Ft) |
| 隔震設計位移用彈性剛度 | 需用等效割線剛度 Keff，非初始剛度 |

## 高頻考點

- 橋梁週期公式（α 法）：幾乎每年考
- 建築基底剪力與樓層分布：高頻
- 防震間距（SRSS 組合）：相鄰建築不同頻率
