# MDOF 題型診斷

> 適用：`SD-U1-3`（多自由度系統模態分析）

## 快速識別

| 題目特徵 | 判斷 |
|---------|------|
| 給多樓層質量矩陣 [M]、勁度矩陣 [K] | 標準 MDOF 特徵值題 |
| 給「柔性基座」、「旋轉彈簧 kθ」 | SSI 模型，注意耦合自由度 |
| 給連續梁 + 假設振型 | Rayleigh-Ritz 廣義自由度法 |
| 問「模態疊加」後地震反應 | 需求 Γᵢ、SRSS/CQC |

---

## 決策樹

```
MDOF 題
  │
  ├─ 集中質量剪力構架（最常見）
  │    ├─ 建立 [M]（對角）與 [K]（三對角）
  │    ├─ det([K] - ω²[M]) = 0 → 特徵值 ω₁², ω₂²,...
  │    ├─ 代回 → 振態 {φ}ᵢ（正規化：φ₁ = 1）
  │    └─ 模態參與因子 Γᵢ = {φ}ᵢᵀ[M]{1} / ({φ}ᵢᵀ[M]{φ}ᵢ)
  │
  ├─ SSI 柔性基座
  │    ├─ 識別自由度（旋轉 θ₁ + 水平 u₂）
  │    ├─ 軸向剛性柱 → θ₂ = θ₁ → I₂ 進入 M₁₁
  │    ├─ 用 Lagrangian 建立動能 T 與位能 V
  │    └─ 特徵值問題同上
  │
  └─ Rayleigh-Ritz（連續體）
       ├─ 選假設振型 φ(x)（滿足邊界條件）
       ├─ m* = ∫ρA φ² dx + Σmᵢφ(xᵢ)²
       ├─ k* = ∫EI (φ'')² dx - N∫(φ')² dx（含幾何勁度）
       └─ ω₁² = k*/m*（上界定理）
```

## 2-DOF 等質量等勁度剪力構架（高頻考點）

$$\lambda^2 - 3\lambda + 1 = 0 \Rightarrow \lambda_{1,2} = \frac{3 \pm \sqrt{5}}{2}$$

- 振態 1：$\{1,\ 1.618\}$（黃金比例）
- 振態 2：$\{1,\ -0.618\}$

## 有效質量驗算

$$\sum m_i^* = m_1^* + m_2^* = \text{總質量}$$

若有效質量比 < 90%，需納入更多模態。

## 常犯錯誤

| 錯誤 | 正確作法 |
|------|---------|
| SSI 題中忽略 I₂ | 軸向剛性柱使 θ₂=θ₁，I₂ 必須進入 M₁₁ |
| Rayleigh 幾何勁度項正負號 | 軸壓降低有效勁度：k* = k_彈性 - k_幾何 |
| 振態未正規化就計算 Γᵢ | 任意正規化方式均可，但需一致 |
